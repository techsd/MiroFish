# Changelog

Усі помітні зміни в цьому проекті будуть задокументовані в цьому файлі.

Формат базується на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), і цей проект дотримується [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased] - 2026-04-29

### Додано (Added)
- **Ukrainian Localization (Українська локалізація)**: 
  - Повна підтримка української мови для інтерфейсу багатоагентної системи (органічний переклад понад 660 ключів у `locales/uk.json`).
  - Додано конфігурацію української мови у `locales/languages.json` з відповідними інструкціями для LLM.
  - Створено повноцінний україномовний посібник `README-UA.md`, адаптований з акцентом на практичне застосування (маркетингові дослідження, аналіз громадської думки) та автономність системи.
  - Оновлено навігаційні посилання у `README.md` та `README-ZH.md`.
- **Offline Deployment (100% Автономний режим)**:
  - Створено `OFFLINE.md` з детальним гайдом для повністю автономного розгортання в закритих мережах (включно із zero-config підходами).
  - Написано скрипт автоматичної міграції `offline_migration.py` для безшовної заміни хмарного `zep-cloud` на локальний `zep-python`.
- **Tuning & Troubleshooting (Налаштування та вирішення проблем)**:
  - Створено `TUNE.md` з інструкціями щодо переналаштування проектних портів, вибору та конфігурації локальних LLM-моделей (LM Studio, Ollama).
  - Створено `FIX.md` для усунення типових помилок (наприклад, проблем з кодуванням на ОС Windows, таймаутами при тривалих симуляціях та оптимізації Docker-збірки).

### Змінено (Changed)
- **Dependency & SDK Updates (Оновлення залежностей)**:
  - Розпочато процес переходу бекенд-залежностей із закритих хмарних сервісів (`zep-cloud`) на локальні відкриті альтернативи (`zep-python`).
  - Підготовлено оновлення для файлів `requirements.txt` та `pyproject.toml`, щоб забезпечити локальне середовище виконання.
- **Documentation Sanitization**:
  - Документація була очищена від зайвих emoji для досягнення більш професійного та суворого (production-ready) стилю.
